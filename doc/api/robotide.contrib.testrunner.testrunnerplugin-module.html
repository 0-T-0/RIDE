Module testrunnerplugin

source code

A plugin for running tests from within RIDE

Some icons courtesy Mark James and provided under a creative commons license. See http://www.famfamfam.com/lab/icons/silk

Note: this plugin creates a temporary directory for use while a test is running. This directory is normally removed when RIDE exists. If RIDE is shut down abnormally this directory may not get removed. The directories that are created match the pattern RIDE*.d and are in a temporary directory appropriate for the platform (for example, on linux it's /tmp).

You can safely manually remove these directories, except for the one being used for a currently running test.
