RIDE -- Robot Framework test data editor

Usage: ride.py [inpath] [outpath]

RIDE can be started either without any arguments or by giving a path to a test data file or directory to be opened. RIDE can also be used for 'tidying' Robot Framework test data by giving both inpath and outpath arguments. Tidying works for both test case and resource files, but it does not work with test data directories. When RIDE is used like this, no GUI is opened.

RIDE's API is still evolving while the project is moving towards the 1.0 release. The most stable, and best documented, module is robotide.pluginapi.
