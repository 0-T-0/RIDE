copy(self)

source code 
Copy the contents of the selected cell(s). This does a normal copy action if the user is editing a cell, otherwise it places the selected range of cells on the data.

cut(self)

source code 
Cuts the contents of the selected cell(s). This does a normal cut action if the user is editing a cell, otherwise it places the selected range of cells on the clipboard.

paste(self)

source code 
Paste the contents of the clipboard. If a cell is being edited just do a normal paste. If a cell is not being edited, paste whole rows.
