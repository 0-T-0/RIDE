Module runprofiles

source code

runProfiles.py

This module contains profiles for running robot tests via the runnerPlugin.

Each class that is a subclass as BaseProfile will appear in a drop-down list within the plugin. The chosen profile will be used to build up a command that will be passed in the tests to run as well as any additional arguments.
