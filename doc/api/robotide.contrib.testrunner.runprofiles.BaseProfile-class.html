Class BaseProfile

Base class for all test runner profiles

At a minimum each profile must set the name attribute, which is how the profile will appear in the dropdown list.

In case some settings are needed, provide default_settings class attribute with default values.

This class (BaseProfile) will _not_ appear as one of the choices. Think of it as an abstract class, if Python 2.5 had such a thing.
