                         RunAnything

A plugin for executing commands on the system.

This plugin enables creation of persistent run configurations and execution of those. Output of the executed command is displayed in a separate tab.

Method Details

__init__(self, app)
(Constructor)

source code 

Initialize the plugin with the provided data.

The provided information is mainly used by the plugin manager. Simple plugins are often fine with the defaults. If this method is overridden, the plugin must call it explicitly:

from robotide.pluginapi import Plugin

class MyCoolPluginExample(Plugin):
    """This extra cool docstring is used as the plugin doc."""
    def __init__(self, application):
        Plugin.__init__(self, application, metadata={'version': '0.1'},
                        default_settings={'color': 'red', 'x': 42})

Plugins should not create any user interface elements at this point but wait until the enable method is called.

Parameters:
  • application - RIDE application reference.
  • name - Name of the plugin. If not specified, the name is got from the plugin class name dropping possible Plugin from the end.
  • doc - Plugin documentation. If not specified, the doc is got from the plugin class docstring.
  • metadata - A dictionary of free metadata shown on the plugin manager. Values containing URLs will be shown as links.
  • default_settings - A dictionary of settings and their default values. Settings are automatically stored onto RIDE configuration file, can be accessed using direct attribute access via __getattr__, and new settings can be saved using save_setting.
  • initially_enabled - Specifies should the plugin be enabled when loaded for the first time. Users can change the status later from the plugin manager.
